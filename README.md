# Homepage — резюме

Одностраничное резюме: два параллельных роадмапа (образование и карьера), навыки, увлечения, контакты. Цель — работодатель или потенциальный менти.

## Как открыть

Просто открой файл `index.html` в браузере (двойной клик или «Open with»). Можно поднять локальный сервер:

```bash
# Python 3
python3 -m http.server 8000

# или npx
npx serve .
```

После этого открой в браузере: http://localhost:8000

## Что править

- **index.html** — имя, контакты (email, Telegram, GitHub, LinkedIn), тексты в hero, названия компаний/вузов, годы, описания проектов в карточках, блок «Увлечения и обо мне», ссылки в футере.
- **styles.css** — при желании поменять цвета в `:root` (например `--accent`, `--accent-education`, `--accent-career`).
- Шрифты подгружаются с Google Fonts (Unbounded, JetBrains Mono). При работе офлайн они не загрузятся — можно заменить на системные или локальные.

## Структура

- Hero с навигацией по якорям.
- Два роадмапа рядом (на узком экране — друг под другом): образование и карьера, с таймлайнами и карточками проектов. «Подробнее» раскрывает описание и проекты.
- Блок навыков (языки, инфра, домен).
- Блок «Увлечения и обо мне».
- Контакты и призыв связаться.

Файлы: `index.html`, `styles.css`, `script.js`. Никакой сборки не требуется.
